Class CurlHttpLib

Description
Description | Methods (details) Constants (details)

Implements interfaces:

  • OpenSocialHttpLib

An implementation of the OpenSocialHttpLib interface that uses curl.

This implementation should be more robust than the sockets implementation but requires curl to be built into PHP.
